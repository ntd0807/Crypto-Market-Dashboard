\# 📊 Crypto Market Dashboard



\## 🚀 Overview



This project builds an automated data pipeline and interactive dashboard to analyze cryptocurrency market trends.



It integrates:



\* Python (data ingestion)

\* CoinMarketCap API (data source)

\* Tableau Public (visualization)



\---



\## 🔧 Features



\* Automated data extraction using CoinMarketCap API

\* Data cleaning and transformation with Pandas

\* Historical data storage in CSV format

\* Interactive Tableau dashboard including:



&#x20; \* 📈 Price Trend Over Time

&#x20; \* 📊 Top Movers (24h)

&#x20; \* 💰 Market Cap Ranking

\## ⚙️ How It Works



1\. Python script pulls cryptocurrency data from API

2\. Data is cleaned and appended to CSV file

3\. Tableau reads CSV and visualizes trends

4\. Dashboard provides interactive filtering and analysis

\## 🧠 Key Insights

\* Mid and low-cap coins exhibit higher short-term fluctuations

\* Market dominance is heavily concentrated among top-ranked assets
